# 调试命令

## ETH

查看ETH是否开启

```sh
cat /sys/class/net/eth0/carrier
cat /sys/class/net/eth0/operstate

ifconfig eth0 down
ifconfig eth0 up
ifconfig eth0 192.168.50.247

mkdir /dev/pts; mount -t devpts none /dev/pts;
telnetd &
```
